Show me how this layout shrinks when you zoom in.
exp1a · how to reproduce
On Chrome mobile: pinch-zoom in. The layout height updates because visualViewport.resize fires with a smaller height value, and --app-height is set to visualViewport.height regardless of scale. The whole chat viewport shrinks.
