How does this fix the zoom issue?
exp1b · explanation
The scale guard checks visualViewport.scale before updating --app-height. If scale > 1 (user has zoomed), we skip the update. This means keyboard show/hide still resizes the layout, but pinch-zoom does not. The layout stays stable while zoomed.
